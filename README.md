 Advanced Currency Converter

A modern and responsive Currency Converter web application that supports 195+ world currencies with real-time exchange rates.

roject Overview

This web-based application allows users to convert currencies instantly using live exchange rate data. It features a clean UI, dark/light theme toggle, country flags, and detailed currency information.

Features

- Supports 195+ global currencies  
- Real-time exchange rate conversion  
- Dark & Light theme toggle  
- Automatic country flag display  
- Detailed currency information section  
- Currency swap functionality  
- Amount in words conversion  
- Local storage caching for exchange rates  
- Fully responsive design  

  Technologies Used

- HTML5  
- CSS3 (Responsive Design + Media Queries)  
- JavaScript (ES6+)  
- ExchangeRate API (Public endpoint)  
- LocalStorage for caching  

 API Usage

This project uses a public exchange rate API endpoint and does not require or expose any API key.

Learning Outcomes

- API integration using Fetch  
- DOM manipulation  
- Responsive UI design  
- Debouncing user input  
- Error handling & caching logic
